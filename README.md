## 🚀 Deploy on GitHub Pages

You can host this portfolio for free using GitHub Pages.

### Step 1 — Push the files to your repository
Make sure these files are in the root of the repo:

- `index.html`
- `projects.html`
- `certificates.html`
- `contact.html`
- `styles.css`
- `script.js`

### Step 2 — Enable GitHub Pages
1. Open the repository on GitHub
2. Go to **Settings**
3. Click **Pages**
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Save the settings

### Step 3 — Open the live site
After deployment, GitHub will generate your portfolio URL.

If the repo name is:
- `username.github.io` → site opens at the root domain
- `portfolio` or another custom name → site opens at `/repository-name/`

## 🚀 Deploy to GitHub Pages

### Step 1 — Create repository
```bash
# Create a repo named:  your-username.github.io
# OR any name like "portfolio"
```

### Step 2 — Add files and push
```bash
git clone https://github.com/Mohit-Dagar/Mohit-Dagar.github.io.git
cd Mohit-Dagar.github.io

# Copy: index.html, projects.html, certificates.html,
#       contact.html, styles.css, script.js

git add .
git commit -m "🚀 Launch portfolio"
git push origin main
```

### Step 3 — Enable GitHub Pages
1. Repo → **Settings** → **Pages**
2. Source: `main` branch → `/ (root)` folder
3. Click **Save**
4. Visit: `https://Mohit-Dagar.github.io` in ~2 minutes ✅

## ⚙️ Add Real Email (Formspree)

1. Sign up at [formspree.io](https://formspree.io) → create a form
2. Copy your endpoint (e.g. `https://formspree.io/f/abcdefgh`)
3. In `contact.html`, change the `<form>` opening tag to:
```html
<form class="contact-form" action="https://formspree.io/f/YOUR_ID" method="POST">
```
4. In `script.js`, remove or comment out the `e.preventDefault()` line inside `initContactForm()`
